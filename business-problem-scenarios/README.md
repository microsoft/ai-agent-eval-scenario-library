# Business-Problem Scenarios

Evaluation scenarios grounded in **what your agent solves for users**. These scenarios map to common agent purpose patterns — not specific industries or agent types, but the type of business problem being solved.

## Why Start Here

Customers build agents to solve business problems. These scenarios let you evaluate whether your agent delivers the right business value, regardless of the underlying infrastructure.

## Scenarios

| Business-Problem Group | What It Covers | Link |
|----------------------|---------------|------|
| **Information Retrieval & Policy Q&A** | Agent answers questions by retrieving from authoritative sources. Tests accuracy, currency, personalization. | [View scenarios](information-retrieval-and-policy-qa.md) |
| **Troubleshooting & Guided Diagnosis** | Agent walks users through diagnostic steps. Tests step accuracy, platform awareness, escalation judgment. | [View scenarios](troubleshooting-and-guided-diagnosis.md) |
| **Request Submission & Task Execution** | Agent takes action on behalf of the user. Tests correct execution, confirmation, error handling. | [View scenarios](request-submission-and-task-execution.md) |
| **Process Navigation & Multi-Step Guidance** | Agent guides users through multi-step processes. Tests completeness, sequence accuracy, role awareness. | [View scenarios](process-navigation-and-multistep-guidance.md) |
| **Triage & Routing** | Agent determines the right destination and hands off. Tests classification accuracy, context preservation. | [View scenarios](triage-and-routing.md) |

## How These Connect to Capability Scenarios

Business-problem scenarios test the **what** — is the agent delivering correct answers, executing the right actions, guiding users properly? But they don't test the **how** — is it using the right knowledge sources, calling tools with correct parameters, staying within safety boundaries?

After selecting your business-problem scenarios, browse the [capability scenarios](../capability-scenarios/) to cover infrastructure, compliance, and safety dimensions.
